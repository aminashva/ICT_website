Robotics project

Our work with LEGO Mindstorms EV3 robot started with the exploration of the Robot Trainer tutorials. Next, we started writing
a program for the robot, taking all the requirements into the consideration.
We had to think of all the possible scenarios that may take place, due to the fact that we do not have an
actual robot, so it is impossible to test the program beforehand. After some investigation, we came to the conclusion
that robot works by the following principle: robot turns either left
or right based on the reflected light
intensity, which is measured by the color sensor.
