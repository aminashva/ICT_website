Project description:

The objective of the project was to create a website with a use HTML, CSS, and JavaScript programming languages. Firstly, in order
to understand how websites work and how to create one, each member of the team took HTML,
CSS and JavaScript courses provided my Codecademy. After the successful completion of these courses, our team started
to actively work on the development of our website. We started by building a skeleton of the website with the use of HTML. Next, we started to work on
the design of the website, using CSS. Finally, with the use of JavaScript,
some final details were added to the website.
